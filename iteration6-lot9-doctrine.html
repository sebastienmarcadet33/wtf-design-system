Doctrine de bascule chromatique.

La bascule entre territoires est sèche au routing — le rechargement de route applique une classe .scope-{entite} sur <body> qui redéfinit les variables CSS de l'entité. Pas de transition au scroll, pas de fondu, pas d'animation de bienvenue. La marque change parce qu'on a changé de page.

Une seule règle de bascule par entité. Source unique : tokens-v3.css. 

/* Bascule chromatique · une règle par entité */
.scope-ees {
  --bg:     var(--ees-cream);
  --bg-2:   var(--ees-rose);
  --accent: var(--ees-gold);
}

.scope-ses {
  --bg:     var(--ses-light);
  --accent: var(--ses-gold);
  --accent-2: var(--ses-violet);
}

/* Le footer reste invariant.            */
/* La page peut être EES rose poudré,    */
/* mais le pied dit maison commune.      */
.eco-footer { background: var(--wtf-nuit-chaude); }

03Échelle typographique fluide

Six paliers clamp(). Une seule équation par registre.

L'échelle typographique fluide via clamp() évite les sauts brutaux entre breakpoints. Chaque palier interpole linéairement entre une borne basse (mobile) et une borne haute (desktop large) selon la largeur de viewport. Plancher absolu : 16 px pour le body, 12 px pour les méta — WCAG AA et plancher TIP V3 respectés.

04Interactions

Pas de halo SaaS. Pas d'urgence. Une seule courbe d'easing.

Le registre éditorial impose une grammaire d'interaction sobre. Les transitions existent, mais elles retiennent au lieu de claquer. Une seule courbe d'easing pour tout l'écosystème — la même qu'on retrouverait dans le pas patient d'une visite chez Aesop.

Focus visible. Outline 2px, offset 2px.

Refus absolu du halo SaaS bleu. Le focus visible respecte la palette de l'entité — il prend la couleur --accent du scope actif. Toujours un outline net, jamais un glow. 

:focus-visible {
  outline: 2px solid var(--accent);
  outline-offset: 2px;
}
/* Jamais box-shadow halo, jamais glow */

Sticky. Refus du sticky agressif.

Refusé
  • CTA flottant qui suit le scroll
  • Header qui se réduit en taille au scroll
  • Sidebar de partage social qui colle
  • Bannière cookies persistante anti-respiration
  • Newsletter pop-up déclenchée au scroll
Accepté
  • Header standard sticky sans réduction
  • Sidebar table des matières sur articles longs desktop
  • Skip link "Aller au contenu" focus-only

Curseurs. Default partout. Pointer sur cliquables. C'est tout.

Pas de curseur personnalisé (rond suiveur, magnétisme, blob). Pas de cursor:wait abusif. Pas de cursor:not-allowed décoratif. La main est silencieuse — elle bouge quand il faut, elle disparaît quand il faut.

Huit composants. Trois viewports. Une seule grammaire.

Le library expose les composants atomiques qui structurent les cinq sites. Chacun est démontré à 375 / 768 / 1440 — les trois breakpoints qui font foi. Les viewports sont mis à l'échelle pour cohabiter sur cette double page ; les proportions internes restent fidèles.

06Garde-fous accessibilité

WCAG AA. Tenu par la grammaire, pas posé en surcouche.

L'accessibilité n'est pas une couche d'overlay ajoutée en fin de chantier. Elle est inscrite dans la grammaire même du système — contrastes calculés sur palette, sémantique HTML propre, focus visible respectueux de la palette de l'entité. WCAG AA n'est pas un plafond, c'est un plancher.

Contraste & encres.

Sur fonds clairs, plancher --ink-2 #3B342A (contraste 4.5:1 garanti). Sur fonds sombres, encres crème #E6DDC9. --ink-mute réservé corps ≥ 18px sur fond très clair.

CibleRatio
Texte courant (corps 16px)4.5:1
Texte large (≥ 18px ou ≥ 14px bold)3:1
Éléments UI (focus, boutons)3:1

Sémantique HTML.

  • <nav> · <main> · <article> · <section> · <aside> · <footer>
  • Un seul <h1> par page
  • Hiérarchie h2/h3/h4 respectée — pas de saut
  • Skip link « Aller au contenu » focus-only
  • Tab sequence logique, focus visible toujours
  • Drawer mobile fermable par Escape · FAQ accordéons Enter/Space

Préférences utilisateur. On les écoute, sans surinterpréter.

/* Réduction de mouvement honorée */
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}

prefers-reduced-motion · toutes animations passent à 0.01ms. La grammaire éditoriale reste la même — c'est juste le mouvement qui s'efface.

prefers-color-scheme: dark · refusé en V1. Le registre éditorial impose une palette stable. Le mode sombre dégraderait la cohérence chromatique de l'écosystème (la nuit chaude WTF n'est pas un thème sombre, c'est une palette signature).

07Posture cardinale TIP V3

Les écoles sont des boussoles.
Pas des templates.

Aesop, Hermès, The Row, Frédéric Malle, The Beauty Chef Edit, Pukka — ces maisons cadrent le registre du système TIP V3. Elles ne le composent pas. Ce qu'elles enseignent : sobriété éditoriale, autorité tranquille, refus du SaaS criard, respect du temps de lecture, photographie vraie. Ce qu'elles ne donnent pas : la signature.

07·aCe qui fait la signature unique
Palette double or.

Or terre #B8884A EES/WTF + or canalisé #C9A24E SÉSÂMES/Voie/AVE. Aucune maison existante ne porte cette double signature chromatique. Doctrine de bascule alignée sujet — jamais mixée.

Sceaux iconographiques propriétaires.

Papillon-corps WTF, triangle aquarellé canalisé SÉSÂMES, phénix renaissant AVE, ligature lune-C EES. Iconographie photoréaliste par soustraction — non vectoriellement redessinée, non générique.

Doctrine fondatrice nomade.

Les fondateurs sont les fondations qui voyagent. Aucune palette ne dérive d'un territoire fixe. La maison se porte avec Céline & Sébastien. Différence stratégique majeure rarement explicitée ailleurs.

Articulation 3 chemins + 1 hub publiant.

Inédite. Pas une marque-mère + sous-marques. Pas un parcours linéaire. Trois chemins légitimes, indépendants, qui partagent une voix et une grammaire. Un hub pédagogique discret qui irrigue sans push.

Voix d'incarnation.

« On organise des passages. » Pas « on vend du soin », pas « on coach », pas « on transforme ». La langue dit ce que la pratique fait : tenir l'espace, reconnaître les seuils, nommer ce qui se traverse.

Cinq tensions tenues simultanément.

Profond sans grandiloquence · simple sans pauvreté · clair sans froideur · chaleureux sans mollesse · premium sans arrogance. Aucun maison existante ne tient les cinq d'un coup — c'est précisément ce qui rend la signature reconnaissable.

07·bLe test final · révisé V3.0c

Cette maquette pourrait-elle être évidemment et uniquement un site WTF TIP V3 ?

La grammaire visuelle s'attribue-t-elle d'elle-même à l'écosystème Way To Freedom — et à personne d'autre ?

L'ancien test « Pourrait-elle apparaître comme refonte d'un site Aesop ou Hermès ? » glissait vers le pastiche. Le nouveau test est plus exigeant. Plus juste. Plus aligné avec la posture cofondateurs Marcadet — construire une signature qui dure dix ans et qui n'imite personne.
